**A professional IPTV player for Windows that behaves like a TV set-top box.**

JPTV Player transforms your PC into a living room TV experience with auto-play, fullscreen mode, and keyboard/remote control. Just download, install, and start watching.

### [‚Üí Download Latest Release](https://github.com/ColinGamez/JPTV-Player/releases)

**System Requirements:**
- Windows 10 or Windows 11 (64-bit)
- 4 GB RAM minimum
- 200 MB disk space

**Installation:**
1. Download `JPTV-Player-Setup-x.x.x.exe` from the [Releases](https://github.com/ColinGamez/JPTV-Player/releases) page
2. Run the installer and follow the prompts
3. Launch JPTV Player from your Start Menu
4. On first launch, load your M3U/M3U8 playlist (press **O**)

| Key | Action |
|-----|--------|
| **Arrow Keys** | Navigate channels/categories |
| **Enter** | Play selected channel |
| **ESC** | Close menus, return to playback |

| **E** | Show/hide EPG (program guide) |

| **O** | Open playlist file |
| **L** | Load EPG data (XMLTV file) |
| **Ctrl+D** | Open donation options |
| **Numbers (0-9)** | Direct channel selection |

## üì∫ Using Playlists

JPTV Player requires an **M3U or M3U8 playlist** to function. This file contains channel URLs.

**Where to get playlists:**
- Your IPTV service provider
- Legal, self-hosted streams
- Community-curated lists (verify legality in your region)

**How to load:**
1. Press **O** (or File ‚Üí Open Playlist)
2. Select your `.m3u` or `.m3u8` file
3. Channels will appear immediately

**Sample Playlist:**  
A `sample-playlist.m3u8` file is included in the installation directory for testing.
